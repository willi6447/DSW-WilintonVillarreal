CONCEPTO, DISEÑO Y CONSTRUCCION HTML

    LENGUAJE HTML, DEFINICION Y CONCEPTO

    “HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la siglas que corresponden a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto”

    El lenguaje html (hypertext markup language) se utiliza para el desarrollo y creación de páginas web. Se compone de una serie de etiquetas que el navegador interpreta. Entre las etiquetas que se incluyen dentro del lenguaje HTML se encuentran: hipervínculos, etiquetas para imágenes, saltos de página, entre otras.

INTRODUCCION A LAS HOJAS DE ESTILOS

Las hojas de estilo representan un avance importante para los diseñadores de páginas web,

al darles un mayor rango de posibilidades para mejorar la apariencia de sus páginas.

Las hojas de estilo resuelven estos problemas al mismo tiempo que reemplazan al limitado rango

de mecanismos de presentación del HTML. Con las hojas de estilo es más fácil especificar

la cantidad de espacio entre líneas, el sangrado de las líneas, los colores a utilizar

para el texto y el fondo, el tamaño y estilo de las fuentes, y otros muchos detalles.

Referencia Apa."http://html.conclase.net/w3c/html401-es/present/styles.html#"

HTML5, estructura básica y elementos semánticos

Una de las principales ventajas de HTML5 es la inclusión de elementos semánticos, o marcados semánticos, que nos ayudan a definir las distintas divisiones de una página web.

se puede observar en el esquema de la figura mediante marcadores semánticos se puede diferenciar claramente el contenido que referencia cada etiqueta.
